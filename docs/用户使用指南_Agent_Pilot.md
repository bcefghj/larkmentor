# LarkMentor Agent-Pilot 用户使用指南

> **一句话介绍**：在飞书 IM 中发送 `/pilot <你的需求>`，Agent 自动生成产品方案文档 + 架构图 + 评审 PPT

---

## 🚀 快速开始（3步上手）

### 第1步：添加机器人

在飞书中搜索 **`LarkMentor`**，点击添加为好友

### 第2步：发送指令

私聊机器人，发送：
```
/pilot help
```

### 第3步：体验完整功能

尝试一个完整的工作场景：
```
/pilot 把最近讨论整理成产品方案文档 + 架构图 + 评审PPT
```

**预期效果**：
- ⏱️ 3-5秒后收到执行计划
- 📊 40-60秒自动生成所有产物
- 🔗 最后收到飞书分享链接

## 🔍 常见问题

### Q1：Bot 不响应怎么办？

**可能原因**：
1. Bot 可能正在重启（等待1-2分钟）
2. 权限还在生效中（新申请权限需要5-10分钟）

**解决方法**：
1. 先发送 `帮助` 测试基础功能
2. 如果基础功能正常，再试 `/pilot help`
3. 等待5-10分钟后重试

### Q2：生成的产物在哪里？

**飞书文档**：会自动创建到您的飞书云文档
**架构图和PPT**：在归档消息中点击链接查看
**所有产物**：最后会有一个统一的飞书分享链接

### Q3：可以修改生成的内容吗？

**可以！**
- 飞书文档：直接在飞书中编辑
- 架构图：下载 JSON 后用 tldraw 编辑
- PPT：下载 Markdown 后用 Slidev 编辑

### Q4：支持语音输入吗？

**支持！** 在 Flutter 客户端中：
- 长按「语音输入」按钮录音
- 松开自动转写并触发 `/pilot`

### Q5：执行失败怎么办？

查看执行计划详情页，会显示：
- 失败的步骤
- 错误原因
- 已完成的部分产物

大部分步骤失败不会影响其他步骤继续执行。

### Q6：多端同步是什么意思？

如果您在多个设备登录（如手机+电脑）：
- 执行状态实时同步
- 编辑操作实时同步
- 离线编辑会自动合并
